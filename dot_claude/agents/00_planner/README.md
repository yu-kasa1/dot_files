# 設計部隊 (00_planner)

要件定義から仕様策定、タスク分解までを担当するエージェント群。

## エージェント一覧
| エージェント | 役割 |
|-------------|------|
| requirement-definer | ユーザーの要望を整理し、要件を定義する |
| spec-writer | 要件から詳細仕様を策定する |
| task-writer | 仕様からタスク一覧を作成する |
| code-reviewer | 設計レビューを実施する |

## 利用可能なユーティリティ
`@99_utility` 配下のエージェントも必要に応じて呼び出し可能:
- `@db-analyzer`: DB構造の調査、テストデータ生成
